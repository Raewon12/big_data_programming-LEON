| `Class_info.class_name` | 사건 유형 | 분류 라벨 (정답) |

#### 1-1. JSON → CSV 변환
- 66,511개 JSON 파일이 든 폴더 경로를 지정
- 반복문으로 전부 열어서 `jdgmn`, `summ_pass`, `class_name` 세 필드만 추출

```python
import os, json, pandas as pd

data = []
folder = '라벨링데이터_폴더_경로'

for filename in os.listdir(folder):
    if filename.endswith('.json'):
        with open(os.path.join(folder, filename), 'r', encoding='utf-8') as f:
            d = json.load(f)
        try:
            row = {
                'jdgmn': d['jdgmn'],
                'summ_pass': d['Summary'][0]['summ_pass'],
                'class_name': d['Class_info']['class_name']
            }
            data.append(row)
        except (KeyError, IndexError):
            continue  # 필드 없는 파일은 스킵

df = pd.DataFrame(data)
df.to_csv('legal_cases.csv', index=False)
print(f"총 {len(df)}건 저장 완료")
```

```python
def clean_text(text):
    text = re.sub(r'\n+', ' ', text)          # 줄바꿈 → 공백
```

### STEP 4: BERT 임베딩 + MLP (담당: D)

> **중요: STEP 1이 끝나야 시작 가능. Colab GPU 환경 필요**

#### 4-1. BERT 임베딩 추출
```python
import torch
import numpy as np
import pandas as pd
from transformers import AutoTokenizer, AutoModel
from torch.utils.data import DataLoader

tokenizer = AutoTokenizer.from_pretrained("klue/bert-base")
model = AutoModel.from_pretrained("klue/bert-base")
model.eval()

device = torch.device('cuda' if torch.cuda.is_available() else 'cpu')
model.to(device)

def extract_embeddings(texts, batch_size=32):
    all_embeddings = []
    for i in range(0, len(texts), batch_size):
        batch = texts[i:i+batch_size]
        inputs = tokenizer(batch, return_tensors='pt', max_length=512,
                          truncation=True, padding=True).to(device)
        with torch.no_grad():
            outputs = model(**inputs)
            embeddings = outputs.last_hidden_state[:, 0, :]  # [CLS] 토큰
        all_embeddings.append(embeddings.cpu().numpy())
        if i % 1000 == 0:
            print(f"{i}/{len(texts)} 완료")
    return np.concatenate(all_embeddings)

# 판시사항 임베딩 추출
train = pd.read_csv('train.csv')
train_embeddings = extract_embeddings(train['jdgmn'].tolist())
np.save('train_jdgmn_embeddings.npy', train_embeddings)
print(f"임베딩 shape: {train_embeddings.shape}")  # (약 53000, 768)

# test도 동일하게 추출
test = pd.read_csv('test.csv')
test_embeddings = extract_embeddings(test['jdgmn'].tolist())
np.save('test_jdgmn_embeddings.npy', test_embeddings)
```

**주의**: 66,511건 전체 추출에 Colab GPU로 1~2시간 걸림. 추출 후 .npy로 저장해두면 다시 안 해도 됨

#### 4-2. MLP 분류기 설계
```python
import torch.nn as nn

class LegalClassifier(nn.Module):
    def __init__(self, input_dim=768, num_classes=10):
        super().__init__()
        self.model = nn.Sequential(
            nn.Linear(input_dim, 256),
            nn.ReLU(),
            nn.Dropout(0.3),
            nn.Linear(256, 64),
            nn.ReLU(),
            nn.Dropout(0.3),
            nn.Linear(64, num_classes)
        )
    
    def forward(self, x):
        return self.model(x)
```

#### 4-3. MLP 학습
```python
from torch.utils.data import TensorDataset, DataLoader

# 데이터 준비
X_train = torch.FloatTensor(np.load('train_jdgmn_embeddings.npy'))
y_train = torch.LongTensor(train['label'].values)
X_test = torch.FloatTensor(np.load('test_jdgmn_embeddings.npy'))
y_test = torch.LongTensor(test['label'].values)

train_dataset = TensorDataset(X_train, y_train)
train_loader = DataLoader(train_dataset, batch_size=64, shuffle=True)

# 모델, 손실함수, 옵티마이저
model_mlp = LegalClassifier().to(device)
criterion = nn.CrossEntropyLoss()
optimizer = torch.optim.Adam(model_mlp.parameters(), lr=1e-3)

# 학습 루프
for epoch in range(20):
    model_mlp.train()
    total_loss = 0
    for X_batch, y_batch in train_loader:
        X_batch, y_batch = X_batch.to(device), y_batch.to(device)
        optimizer.zero_grad()
        outputs = model_mlp(X_batch)
        loss = criterion(outputs, y_batch)
        loss.backward()
        optimizer.step()
        total_loss += loss.item()
    print(f"Epoch {epoch+1}/20, Loss: {total_loss/len(train_loader):.4f}")

# 평가
model_mlp.eval()
with torch.no_grad():
    outputs = model_mlp(X_test.to(device))
    _, predicted = torch.max(outputs, 1)
    
from sklearn.metrics import classification_report
print(classification_report(y_test.numpy(), predicted.cpu().numpy()))
```

#### 4-4. 하이브리드 모델 (Feature Fusion)

```python
tfidf = TfidfVectorizer(max_features=5000)

X_test_hybrid = np.concatenate([

# MLP input_dim 수정 (768 + 5000 = 5768)
hybrid_model = LegalClassifier(input_dim=5768).to(device)
# 이후 학습은 4-3과 동일
```

#### BERT + MLP 집중 포인트
- 임베딩 추출은 시간이 오래 걸리니까 **한 번 추출하면 .npy로 반드시 저장**
- 3가지 입력(jdgmn, summ_pass, concat) 각각 임베딩을 따로 추출해야 함 → **총 6개 .npy 파일** (train 3개 + test 3개)
- MLP 구조(256→64→10)는 시작점이고, 성능 보고 **레이어 수나 차원을 조정**해볼 것
- 하이브리드 모델은 BERT 임베딩 + MLP가 완성된 후에 하면 됨
