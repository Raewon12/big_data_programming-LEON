# 사전학습된 ResNet 모델을 활용한 이미지 유사도 검색 보고서

## 1. 과제 개요

ResNet50 사전학습 모델로 강아지 이미지 20장 중 입력 이미지와 가장 유사한 이미지를 찾아내는 시스템을 구현하였다. 마지막 분류층을 제거해 2048차원 임베딩을 추출한 뒤, 코사인 유사도로 가장 비슷한 이미지를 선택하는 방식이다.

## 2. 구현 흐름

1. ResNet50을 불러오고 마지막 분류층(`fc`)을 제거하여 2048차원 임베딩 추출 모델로 재구성
2. 입력 이미지를 224×224로 리사이즈하고 ImageNet 평균·표준편차로 정규화
3. 강아지 이미지 20장에 대해 임베딩 벡터 추출 → `(20, 2048)` 텐서 생성
4. 입력 이미지(`compare.jpeg`)의 임베딩 벡터 추출 → `(1, 2048)`
5. 코사인 유사도 계산 후 가장 높은 값을 가진 이미지 선택

## 4. 분석

ResNet50은 ImageNet으로 사전학습되어 있어 색상이나 배경보다 강아지의 털 패턴, 얼굴 형태, 자세 같은 고차원 특징을 바탕으로 유사도를 판단한다. 실험 결과 같은 품종이나 비슷한 자세의 강아지끼리 높은 유사도를 보였으며, 배경이 복잡하거나 조명이 크게 다를 경우에는 상대적으로 낮은 유사도를 기록하였다.

## 5. 결론

사전학습된 ResNet50을 특징 추출기로 활용하면 별도의 학습 없이도 이미지 간 유사도 기반 검색이 가능하다. 이러한 구조는 이미지 검색, 추천 시스템, 클러스터링 등으로 확장할 수 있다.
